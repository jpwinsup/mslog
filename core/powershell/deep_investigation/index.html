クラッシュなどによるプロセスの異常終了や弊社エンジニアから詳細な情報採取の依頼があった場合

資料採取手順

  1. ログ採取対象の PowerShell を起動します。

  2. 管理者として Cmd を実行し、以下のコマンドによりログ採取対象の PowerShell の PID を確認します。

    tasklist /FI "IMAGENAME eq powershell.exe"

  3. 表示された PID の数値をコピーします。 (例では PID は 61656)

    イメージ名                     PID セッション名     セッション# メモリ使用量
========================= ======== ================ =========== ============
powershell.exe               61656 Console                    2    527,388 K

  4. 以下のコマンドより、資料採取を実行します。

    if not exist C:\MS_DATA mkdir C:\MS_DATA
start ttracer -dumpFull -out C:\MS_DATA -attach <コピーした PID の数値>

  5. クラッシュなど問題が発生する手順を実施します。

  6. 手順 2 で起動した Cmd で以下のコマンドを実行します。

    tttracer -stop *
tttracer -delete *
tttracer -cleanup

  7. C:\MS_DATA に生成されたファイルを ZIP 形式に圧縮し、弊社のエンジニアから案内されたサイトにアップロードします。
