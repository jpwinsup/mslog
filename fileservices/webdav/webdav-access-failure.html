WebDav への接続に関する事象の調査に必要な資料について

いつも弊社製品をご利用いただきまして誠にありがとうございます。
本サイトでは弊社にお問い合わせいただく際に、事前に採取しておいたほうが良い内容についてご紹介しております。
このページでは「WebDav への接続に関する事象の調査に必要な資料について」紹介いたします。

本資料が有効なシナリオ

  • WebDAV サーバーに接続できない
  • WebDAV サーバー上のファイル操作に失敗する

採取いただく資料

  • 作業中のスクリーンショット
  • パケット キャプチャー
  • ファイル共有関連トレース ログ
  • Process Monitor ログ
  • システム情報
    • OS 基本情報 (msinfo32.exe)
    • ネットワーク設定情報 (IP アドレス、ホスト名等)
    • イベント ログ
    • 更新プログラムの適用状況

など

情報採取時の影響について

Process Monitor の採取を含む場合、採取中は一時的にコンピューター リソースへの負荷が高まる可能性がございます。

対象機器

  • 事象が発生するクライアント端末

事前準備

  1. CollectInfo_WebDAV.zip のダウンロード リンクより CollectInfo_WebDAV.zip をダウンロードします。
  2. 採取対象機器に CollectInfo_WebDAV を配置し、展開します。
    ※ 展開後のフォルダーには CollectInfo_WebDAV.bat および t.cmd が配置されていることを確認ください。
  3. Process Monitor - Windows Sysinternals | Microsoft Docs より ProcessMonitor.zip をダウンロードします。
  4. 対象機器に ProcessMonitor.zip を展開し、Procmon.exe を CollectInfo_WebDAV.bat と同じフォルダーに配置します。
    ※ 何らかの事情で Process Monitor の取得が困難な場合は本 “事前準備” の 3. および 4. をスキップしてください。
    ※ Procmon.exe が存在しない場合は自動的に Process Monitor の採取を行いません。

資料採取手順

  1. クライアントに管理者アカウントでログオンします。
  2. CollectInfo-Client.bat ファイルを右クリックし、[管理者として実行] を選択します。
  3. “実行したい番号を入力ください” メッセージに 1 を入力、リターン キーを押下し、表示に従いメニューに戻ります。
  4. 事象を再現します。
  5. “実行したい番号を入力ください” メッセージに 2 を入力、リターン キーを押下し、表示に従いメニューに戻ります。
  6. “実行したい番号を入力ください” メッセージに 3 を入力、リターン キーを押下し、表示に従いメニューに戻ります。
  7. “実行したい番号を入力ください” メッセージに q を入力、リターン キーを押下してツールを終了します。
  8. C:\temp 配下に作成された <YYYYMMDD 形式の年月日>_<ホスト名>_info フォルダーを圧縮し弊社まで提供ください。

本情報の内容（添付文書、リンク先などを含む）は、作成日時点でのものであり、予告なく変更される場合があります。
